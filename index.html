AR-решения для бизнеса

Мы ИТ-компания из Риги, ориентированная на разработку утилитных и развлекательных AR/ML продуктов.

BuduAR

Сервис для создания быстрого и дешевого цифрового медиа-контента для малого и среднего бизнеса. Контент создается с помощью рендера 3Д-одежды, которую пользователи могут сгенерировать на нашей платформе с изображениями продукта на манекене.

Avatarium

«Создай свою цифровую копию». Сервис позволяет пользователям сгенерировать похожего на себя 3Д-аватара с помощью алгоритмов искусственного интеллекта.

StripAR

AR-стриптиз. Мобильное приложение для оставшихся в одиночестве молодых людей.
